# SideLetter
字母侧边栏和recyclerview进行联动

效果图可以去这个链接看看：https://www.jianshu.com/p/5bf306e2c301

第一步 在project build文件中添加

        allprojects {
          repositories {
                maven { url 'https://jitpack.io' }
                }
        }
 第二步 添加依赖库
  
     api 'com.github.wangqiux:SideLetter:v1.0.2'
